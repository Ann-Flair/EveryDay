[75\. Sort Colors(Medium)](https://leetcode.cn/problems/sort-colors/)
## 题解
``` java
class Solution {
    /**
        方法：
            单、双指针 O(N) O(1)
            冒泡排序等排序方法
     */
    //法1：单指针
    public void sortColors(int[] nums) {
        int len = nums.length;
        int pos = 0;
        for (int i = 0; i < len; i++) {
            if (nums[i] == 0) {
                swap(nums, i, pos++);
            }
        }
        for (int i = 0; i <len; i++) {
            if (nums[i] == 1) {
                swap(nums, i, pos++);
            }
        }
    }

    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```
