关键：

``` java
// 时空：O(logN)、O(1)
public int researchBinary(int[] nums, target) {
  int right = nums.length - 1, left = 0;
  while(left <= right) {
    int mid = left + （right - left) / 2; //或 (left + right) / 2 或 (left + right)>>1 或 left + （right - left + 1) / 2向上取整
    if(nums[mid] == target)
      return mid;
    else if(nums[mid] < targrt)
      left = mid + 1;
    else
      right = mid - 1;
  }
  return left;
}
```
